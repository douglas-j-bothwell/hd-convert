Currently, this feature is in Beta.

Harness Git Experience integrates your Harness Projects, Pipelines, and resources with your Git repos. You can work entirely from Git or use a hybrid method. With Harness Git Experience, your Git repos are always the single source of truth.

If you are new to Harness Git Experience, read Harness Git Experience Overview and try the Harness Git Experience Quickstart.
