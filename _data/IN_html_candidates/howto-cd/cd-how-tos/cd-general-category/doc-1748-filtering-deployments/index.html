Visual Summary

You can access the New Filter from Pipelines or Deployments page. Open New Filter by clicking the icon in the upper right hand corner of your screen.

Here is what the Deployments page filter options look like:

As you can see, you can filter according to multiple criteria. Next, you can give your filter a name and save it.

A filter can be private or available for viewing by everyone else.

Option 1: Filter by Pipeline

To filter by Pipeline, enter the exact name of the Pipeline in the Name field and click Apply. The Pipeline that meets the name criteria appears.

The Description field is optional.

You can also enter the Tag associated with the Pipeline.

Option 2: Filter by Deployment

To filter by Deployment Type, select the deployment and click Apply. Additionally, you can select from Services and Environments to include them with the filter.

Option 3: Filter by Statuses

In Status, select all of the statuses you want to filter on.

Option 4: Filter by Build Type

In Build Type, select the type of build by which you want to filter.

Step: Save a Filter

You can save a filter for a Pipeline or a Deployment that meets a specific criterial. For example, you can specify Kubernetes for Deployment Type and save that filter for future use. The saved filter is now available in the list of filters.
