Dashboards help you model and analyze business metrics and operational data. You can use this data to make data-driven business decisions. Using charts, data tables, and filters, Dashboards help you get the most useful deployments data.

Harness provides pre-loaded By Harness (pre-defined) and Custom (user-defined) Dashboards to visualize deployments data across your Applications, Services, and Environment. Using the CG Dashboard you can:

  • Discover new analytical insights into your deployment frequency per application
  • Track deployment status across different zones and time range in a logical and structured manner
  • View your deployments at a glance, understand the applications with the highest failure rate, and analyze deployment trends, etc.

This topic describes how to view the By Harness CG Dashboard and get more information about that data.

Step: View CG Dashboard

Perform the following steps to view CG Dashboard:

  1. In Harness First Generation, click Custom Dashboards Next Gen.
  2. In Dashboards, select By Harness and click CG Dashboard.The CG Dashboard is displayed.
  3. Select Reporting Period to filter the data based on pre-defined time range filters. The available filters are:
    • Last 1 Day
    • Last 7 Days
    • Last 30 Days
    • Last 90 Days
    • Last year
  4. Once you have selected the Time Range filter, click Update. The data is refreshed with the latest data from the database.
