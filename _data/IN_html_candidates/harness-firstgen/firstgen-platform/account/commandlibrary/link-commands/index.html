The Harness Command Library stores custom Service commands that can be shared between all your Harness account users. You can download these commands and link them to a Service or Workflow. You can use the same command in multiple Workflows and Services by linking it as needed.

Service commands are used in the following Service types only:

  • Secure Shell (SSH)
  • WinRM

Link Commands to a Service

To link a command to a Service, do the following:

  1. Open a Service.
  2. In Deployment Specification, click Add Command. The Add Command settings appear.
  3. Select Select from Shared Template Library.
  4. In Downloads, click Link next to the command that you want to link. The command is added to the Service and linked.

Link Commands to a Workflow

To link a command to a Workflow, do the following:

  1. Open a Workflow.
  2. Click Add Step, and then select Template Library.
  3. In Downloads, click Link next to the command that you want to link. The command is added to the Workflow and linked.
