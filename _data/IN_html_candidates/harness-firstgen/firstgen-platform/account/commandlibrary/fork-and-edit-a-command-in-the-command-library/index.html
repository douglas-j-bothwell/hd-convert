Your Services and Workflows can link to Service Commands in the Command Library, but these commands are immutable. To edit them, you need to fork them. Once a Command Library command is forked, it is treated just like a regular Service Command template in the Template Library.

Before You Begin

  • Harness Command Library Overview
  • Add Commands from the Command Library

Step: Fork and Edit a Command

To fork and edit a command, do the following:

  1. Select the command that you want to fork.
  2. Select the Version of the command and click Fork.
  3. Enter the Command Name. Use a name that describes the purpose of the command.The forked command is listed in All Templates in the Template Library.
  4. Click on the template to edit it.
