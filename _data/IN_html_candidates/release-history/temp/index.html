Minor Release 58401

Release Date: Sept 22, 2020

We are pleased to present Harness On-Premise Minor Release 58401.

New Features and Enhancements

The following new features were added to the Harness platform components:

  • FetchUserByEmail API (PL-12430)
    • You can now search users via the Harness API, including those who have not accepted the invite.
      See Fetch Users By Email Address.
  • Make Terraform Plan available outside of Terraform Provisioner (CDP-14334)
    • You can now run the Terraform Apply and Terraform Destroy steps as Terraform plans. See Using the Terraform Apply Command and Remove Provisioned Infra with Terraform Destroy.

  • Customer Unable to Query for UserGroups Using GraphQL API (PL-12495)
  • Create Widget Button is not Working for all the Custom Widgets Except Deployments Widgets (DX-1908)
  • Custom Dashboard: Search is not Getting Applied After Deleting (DX-1847)
  • Second Group by Should not Show First Group by (DX-1142)
  • Variables ${ami.NewAsgName} and ${ami.OldAsgName} not Being Evaluated (CDP-14773, ZD-11375)
  • Fix Typo in Harness Env Variable for PCF: HARNESS_STATUS_INDENTIFIER (CDP-14519, ZD-11240)
  • Showing More Service Instances Than Deployed to Environments (CDP-14469, ZD-10089)
  • ClassCastException: PCF Deployment Fails With UNKNOWN_ERROR (CDC-9330, ZD-11061)
  • Notification Strategy User Expression Config is Getting Auto Deleted (CDC-9092, ZD-10845, ZD-11048)
  • Templatised Infra Field in the Pipeline Execution Step Does not Work Appropriately when Services are Scoped (CDC-8900)
  • Can't Submit Custom Artifact Source Script (CDC-6599)
  • Workflow Breadcrumb Adding Workflow Name to URL Before Workflow Loading (CDC-838)
