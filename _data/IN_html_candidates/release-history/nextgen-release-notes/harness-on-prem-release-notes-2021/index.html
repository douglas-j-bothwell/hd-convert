Issues Fixed in the Release

    • You can now use Git with a generic Git Connector via SSH. This will help you:
      • Use SSH to connect git providers such as Gerrit.
      • Use connection string in format such: ssh://admin@192.168.1.40:29418/All-Projects
      • See Add SSH Keys.

    • Fixed the bug to allow using UserName as secret in Docker Connector.
