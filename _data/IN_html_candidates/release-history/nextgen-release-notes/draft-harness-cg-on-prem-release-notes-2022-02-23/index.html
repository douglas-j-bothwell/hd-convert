Minor Release 73807

Release Date: Feb 23, 2022

We're pleased to present Harness On-Premise Minor Release 73807.

New Features and Enhancements

The following new features were added to the Harness Next Gen platform components:

Platform
  • N/A
CD
  • You can now use Kustomize patches to customize your deployment and enhance it with Harness variables support. This enables you to:
    • Add or replace any value in a Kustomize manifest specific to your Harness deployment without the need to have another layer.
    • Override manifest values in Kustomize based on your service or environment.
    • Use Harness variables in your Kustomize deployment.
    Currently, this feature is behind the Feature Flag KUSTOMIZE_PATCHES_CG. See Option: Kustomize Patches. (CDS-13828)
  • SSH and WinRm deployments now enable you to edit the list of hosts to configure by entering the hostnames in plain text. Previously, the Configure Select Nodes > Host Name(s) field only allowed hosts that were already included in the Infrastructure Definition. (CDS-8075)
    Currently, this feature is behind the Feature Flag DEPLOY_TO_INLINE_HOSTS. See Option: Select Host Not in Infrastructure Definition .
