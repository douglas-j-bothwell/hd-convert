What's New

  • The Security Testing Overview page enables you to see the daily distribution of all issues detected by all STO scans over the past 30 or 90 days.
  • The Security Testing Dashboard enables you to drill down into the set of all security issues detected by all Harness pipeline scans. You can view detected issues by time, criticality, scanner, and number of occurrences. To view all issues by group, click in a chart element. You can also download issue lists to CSV.To view this dashboard, click Dashboards (left menu) and then Security Testing Dashboard. (STO-???)
  • This release include a new set of output variables for builds with security steps. When the build scans a target that differs from the baseline, the following variables show the number of issues found in the target but not in the baseline:
    • NEW_CRITICAL
    • NEW_HIGH
    • NEW_MEDIUM
    • NEW_LOW
    • NEW_UNASSIGNED (Reserved for future use)
    • NEW_TOTAL (CI-???)
